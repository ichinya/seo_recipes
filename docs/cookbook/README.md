# Рецепты

В этом разделе собраны практические инструкции по настройке сайтов, CMS, фреймворков и рекламных или аналитических интеграций.

## Разделы

| Раздел | Что внутри |
| --- | --- |
| [MediaWiki](./mediawiki/) | Sitemap, Яндекс Метрика, рекламные блоки, Sape, Turbo-страницы и связанные настройки |
| [WordPress](./wordpress/) | Практические заметки по WordPress и плагинам |
| [Laravel](./laravel/) | Рецепты и заметки по Laravel |
| [Сайты на Vue](./vue/) | VuePress, Nuxt Content, Метрика и настройки для Vue-проектов |

## Серверные инструкции

Серверные скрипты и Linux-инструкции перенесены в раздел [Хостинг / Скрипты](../hosting/scripts/), чтобы не смешивать рецепты для сайтов с инфраструктурными материалами.
