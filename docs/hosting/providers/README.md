# Провайдеры

Здесь собраны отдельные страницы по хостингам, облакам и VPS/VDS-провайдерам. Провайдеры разделены на четыре практические группы: что можно рекомендовать, что выглядит нормальным с оговорками, кандидаты на тест и рискованные или спорные провайдеры.

Иконки в списке провайдеров:

- `fa-circle-check` - хороший / рабочий вариант;
- `fa-server` - кандидат на тест;
- `fa-triangle-exclamation` - сомнительный или рискованный вариант.

## Рискованные / спорные провайдеры

| IONOS | Крупный немецкий VPS-провайдер: говорят, что из РФ еще работает, но официальная санкционная политика делает аккаунт и оплату высоким риском | [Заметка про IONOS](./ionos.md) |

| PQ.Hosting | Высокий риск: промо после KYC уже менее важно, главный вопрос - прозрачность провайдера и реальная локация VPS | [Заметка про PQ.Hosting](./pq-hosting.md) |
