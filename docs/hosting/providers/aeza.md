# Aeza

Aeza по личному опыту не понравилась: были вопросы к реальной скорости, стабильности серверов и работе сайта.

## Что не понравилось

- заявленная скорость была 1 Гбит/с, а фактически получалось примерно 200-300 Мбит/с;
- субъективно скорость была небольшой, хотя Speedtest мог показывать нормальные результаты;
- пару раз сервер не работал по несколько дней;
- компенсация была только за дни простоя;
- сайт и панель работали нестабильно.

## Про тесты скорости

По Aeza отдельно важно не полагаться только на Speedtest. В личном опыте Speedtest показывал нормальные значения, но реальная скорость в рабочих сценариях все равно не устраивала.

Для проверки такого провайдера лучше смотреть не только синтетический тест, но и реальную задачу: загрузку файлов, работу прокси, задержку до нужных клиентов, стабильность канала в разное время и поведение под длительной нагрузкой.

## Риски

Главный риск - непредсказуемость. Если сервер нужен для рабочих проектов, простой на несколько дней может быть дороже любой экономии на тарифе.

## Итог

Aeza пока не выглядит хорошим вариантом для важных проектов. Можно рассматривать только для задач, где не критичны простои, реальная скорость и качество панели управления.
