# CloudVPS.by

CloudVPS.by стоит добавить как белорусского кандидата на тест для VPS, cloud VPS и обычного хостинга. Личного опыта эксплуатации в этой заметке пока нет, поэтому выводы ниже основаны на публичных страницах провайдера и быстрой проверке сайта 22 июня 2026 года.

## Что известно

По открытым страницам CloudVPS.by на 22 июня 2026 года:

- провайдер работает от имени ООО "Единые Решения", офис и юридический адрес указаны в Минске;
- на сайте есть виртуальный хостинг, обычные VPS/VDS, cloud VPS, выделенные серверы, GPU, домены и дополнительные услуги;
- в разделе "О компании" сказано, что сервис развивают 8 лет, а часть услуг активируется автоматически максимум за 3 минуты;
- на странице cloud VPS заявлены выделенный IPv4 для каждого сервера, NVMe, запуск примерно за 1 минуту, Intel Xeon Gold с частотой до 5,5 ГГц, бесплатные еженедельные бэкапы на тарифах до 200 ГБ, общая емкость сети около 10 Гбит/с и ширина канала до 200 Мбит/с;
- на этой же странице отдельно упомянуты до 5 ТБ бесплатного трафика, приватная сеть между виртуальными машинами и 7 дней теста для юридических лиц и ИП по запросу в отдел продаж;
- отдельные страницы есть для локаций Беларусь, Россия и Европа;
- на виртуальном хостинге заявлены ежедневные бэкапы, бесплатные SSL Let's Encrypt, CloudLinux, Git, Apache + Nginx и AI-BOLIT.

По странице контактов CloudVPS.by:

- по вопросам продаж указан номер `+375 29 635 45 50`;

## Быстрая проверка

Проверка 22 июня 2026 года из текущей сети:

- главная страница `https://cloudvps.by/` открылась с HTTP 200 примерно за 2,0 секунды;
- страница cloud VPS `https://cloudvps.by/servers/cloud/cloud_vps.php` открылась с HTTP 200 примерно за 1,3 секунды;
- страница контактов `https://cloudvps.by/company/contacts.php` открылась с HTTP 200 примерно за 1,0 секунды.

Это только проверка доступности сайта. Реальный сервер не заказывался.

## Плюсы

- белорусский провайдер с публичными реквизитами, адресом и контактной страницей;
- есть несколько типов услуг в одной экосистеме: shared-хостинг, VPS/VDS, cloud VPS, dedicated и GPU;
- заявлен 7-дневный тестовый период;
- на cloud VPS заявлены бесплатный выделенный IPv4, NVMe, резервное копирование и приватная сеть между ВМ;
- на shared-хостинге видны ежедневные бэкапы, Let's Encrypt, Git и CloudLinux;
- есть отдельные страницы под Беларусь, Россию и Европу.

## Минусы и риски

- нет личного опыта эксплуатации;
- по публичным страницам не всегда сразу понятно, какие условия относятся к обычному VPS, а какие к cloud VPS или shared-хостингу;
- условия тестового периода и набор доступных конфигураций нужно перепроверять под конкретный тариф;
- для VPN, прокси, публичных туннелей и других нестандартных сетевых сценариев в этой заметке нет подтвержденной политики провайдера;
- реальную локацию, маршруты, стабильность и скорость из нужных сетей надо проверять отдельным сервером;
- итоговую цену продления, администрирования и дополнительных услуг лучше проверять уже в корзине и переписке с поддержкой.

## Что тестировать перед рекомендацией

- доступность из российских и белорусских сетей, пинг, потери и скорость;
- CPU steal, диск и стабильность VPS под реальной нагрузкой 3-7 дней;
- правила по IPv4, PTR, SMTP, антиабузу и нестандартным сетевым сценариям;
- скорость ответа поддержки через тикеты и отдела продаж до и после заказа;
- реальные условия тестового периода и возврата;
- фактическую цену продления без промо и с нужной панелью или администрированием.

## Итог

CloudVPS.by выглядит как нормальный белорусский кандидат на первичный тест, если нужна локация в Беларуси, BYN-оплата и несколько типов услуг в одной панели. Но пока это именно карточка для проверки, а не рекомендация по личному опыту.
