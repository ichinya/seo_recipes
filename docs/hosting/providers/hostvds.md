# HostVDS

HostVDS пока в разделе как заметка по отзыву коллеги, без личного опыта эксплуатации.

В общей заметке [VPS и блокировки в России, июнь 2026](../info/vps-russia-2026.md) HostVDS оставлен только как неподтвержденный пункт из исходных сообщений: нормального подтверждения связи HostVDS с nLighten / MIRhosting пока нет.

## Контекст

По сообщению Александра Егорова от 8 июня 2026 года, у него используется VPS от HostVDS. По его отзыву, "полет в целом нормальный", а цена выглядит достаточно низкой.

Важный нюанс на момент отзыва: базовые тарифы в наиболее выгодных регионах уже сильно разобраны. Отдельно упоминались Нидерланды / NL как пример региона, где выгодные базовые варианты быстро заканчиваются.

## Плюсы

- VPS работает нормально по отзыву коллеги;
- цены выглядят довольно дешевыми;
- можно рассматривать как кандидата для теста, если нужная локация есть в наличии.

## Минусы и риски

- нет личного опыта эксплуатации;
- базовые тарифы в наиболее выгодных регионах уже сильно разобраны;
- отдельно упоминалась проблема с наличием выгодных тарифов в Нидерландах / NL;
- перед покупкой нужно проверять доступные локации, цену продления и реальные ограничения тарифа.

## Итог

HostVDS выглядит как потенциально недорогой рабочий вариант, но пока это заметка по отзыву коллеги. Главный практический риск - нужные дешевые тарифы в выгодных регионах, особенно в NL, могут быть недоступны.
