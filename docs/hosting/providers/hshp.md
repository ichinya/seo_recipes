# HSHP

[HSHP](https://hshp.host/?from=18970) - вариант для недорогих VPS, когда важна минимальная цена и можно мириться с не самой быстрой поддержкой.

## Контекст использования

Сейчас используется 3 сервера: один для OpenClaw в Финляндии и еще пара в Германии для проксирования.

Есть опыт эксплуатации примерно полтора года в Германии и Финляндии. Даже WireGuard работает нормально.

## Плюсы

- одна из самых низких цен;
- после запуска серверы обычно работают без проблем;
- подходит для задач, где важна низкая стоимость;
- WireGuard на этих VPS работает нормально;
- по опыту коллеги, сервер больше года работал без проблем, кроме одной DDoS-атаки.

## Минусы и риски

- активация иногда бывает долгой;
- техническая поддержка отвечает медленно;
- пару раз отваливался IP: сервер не отвечал, после обращения в поддержку доступ восстанавливали;
- один раз проблему решили примерно за час;
- второй раз проблема была замечена только через 3-5 часов, после этого поддержке понадобилось еще несколько часов на решение;
- для срочных задач лучше заранее учитывать возможную задержку на старте.

## Инциденты

За полтора года эксплуатации были редкие проблемы с доступностью IP. В обоих случаях помогало обращение в техподдержку, но скорость реакции была не моментальной.

Отдельно стоит учитывать риск DDoS: у коллеги за год с лишним заметных проблем не было, кроме одной DDoS-атаки.

## Итог

HSHP можно рассматривать как дешевый рабочий вариант, если сервер не нужен прямо в ближайшие минуты и нет ожидания быстрой поддержки. После запуска, по опыту, серверы обычно работают нормально, но для важных задач нужно учитывать редкие проблемы с IP и небыструю реакцию поддержки.
