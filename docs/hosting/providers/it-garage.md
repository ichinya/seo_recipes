# IT-GARAGE

IT-GARAGE пока в разделе как кандидат на тест VPS. Личного опыта эксплуатации нет, поэтому это не рекомендация, а первичная проверка публичных условий и доступности сайта.

## Что проверено

Проверка 11 июня 2026 года:

- `https://it-garage.pro/index#tariffs` открывается с HTTP 200 примерно за 1.6 секунды;
- `https://my.it-garage.pro/` открывается с HTTP 200 примерно за 1.7 секунды;
- `https://it-garage.pro/js/tariffs.json` открывается с HTTP 200 примерно за 0.3 секунды;
- главная страница отдает тарифы через JavaScript из `js/tariffs.json`;
- остатки тарифов дополнительно обновляются через WebSocket `api.it-garage.pro/ws`;
- DNS в текущей среде отдает reserved/private адреса `198.18.*` и `fc00::*`, поэтому по этой проверке нельзя честно определить реальный IP, ASN и географию.

Это только проверка сайта и публичных условий. Реальный VPS не заказывался.

## Важные ограничения

В оферте и AUP есть условия, которые нужно учитывать до покупки:

- SLA заявлен только 97% в месяц;
- хостинг не гарантирует 100% доступность сети и хост-машин;
- возврат возможен только при неработоспособности сервера по вине хостинга и за фактически неиспользованные дни;
- после приостановки за неоплату данные сохраняются 3 дня, потом могут быть удалены;
- стандартные тарифы гарантируют 25% мощности одного физического ядра на каждый vCPU;
- промо-тарифы гарантируют 10% мощности одного физического ядра на каждый vCPU;
- безлимитный трафик работает с fair use: при чрезмерном использовании скорость могут снизить до 10-100 Мбит/с;
- коммерческие, публичные и массовые туннельные сервисы запрещены;
- личное некоммерческое туннельное использование допускается;
- исходящие SMTP-порты 25 и 465 закрыты по умолчанию;
- на промо-тарифах открыть SMTP-порты 25 и 465 нельзя, PTR/rDNS не предоставляется.

Для туннельных задач это критично: личный WireGuard/Xray как частный доступ по правилам выглядит допустимым, но продавать такой доступ или давать массовый публичный доступ нельзя.

- есть Финляндия от 2.39 евро;
- тарифы и наличие можно проверить через публичный JSON;

- можно платить криптовалютой.

## Минусы и риски

- нет личного опыта эксплуатации;
- Москва на момент проверки в тарифном JSON полностью не в наличии;
- SLA 97% ниже привычных 99.9%;
- у промо-тарифов слабая гарантия CPU: 10% физического ядра на vCPU;
- fair use по трафику может снизить скорость до 10-100 Мбит/с;
- коммерческие публичные туннели запрещены;
- SMTP на промо-тарифах фактически не подходит для почтовых задач;
- внешних отзывов немного, а оценки смешанные;
- юридическое лицо в Великобритании, поэтому это не вариант для задач, где нужен российский провайдер или комплаенс под 152-ФЗ.

## Что тестировать перед рекомендацией

- заказать минимальный тариф в Германии или Финляндии на месяц;
- проверить реальную доступность выбранного тарифа в панели;
- прогнать CPU, диск и сеть минимум 3-7 дней;
- проверить скорость до России, Европы и нужных операторов;
- протестировать личный WireGuard/Xray/Reality, если сервер нужен для частного доступа;
- проверить, нет ли IP в плохих списках и как он проходит трассировки из РФ;
- написать в поддержку до покупки и после создания VPS;
- проверить, как применяются ограничения fair use на длительном трафике.

## Итог

IT-GARAGE выглядит интересным кандидатом для дешевого VPS в Германии или Финляндии, особенно если нужен маленький сервер для тестов или личных задач. Но есть серьезные оговорки: SLA 97%, fair use по трафику, ограниченная гарантия CPU, запрет коммерческих публичных туннелей и отсутствие московских тарифов в наличии на момент проверки.
