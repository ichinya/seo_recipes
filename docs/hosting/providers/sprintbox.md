# Sprintbox

[Sprintbox](https://sprintbox.ru/c5568) - VPS/VDS-направление от ООО "СПРИНТХОСТ.РУ". Его лучше учитывать отдельно от основного shared-хостинга [Sprinthost](https://sprinthost.ru/c5568): Sprintbox - это "боксы", то есть виртуальные выделенные серверы.

## Личный опыт

Сейчас используется один сайт на Sprintbox, примерно полгода. Проблем с ним за это время не было.

Панель немного непривычная, но работает нормально. Техническая поддержка отвечает довольно быстро.

Первичная оценка: вариант дешевый и удобный, но статистики пока мало, потому что в личной эксплуатации пока только один сайт.

## Что известно

По официальной странице, тарифному JSON и панели на 23 июня 2026 года:

- VDS на NVMe;
- бесплатные бекапы и снапшоты;
- заявлен дата-центр Tier III;
- есть API;
- есть установка из базовых образов и своих ISO;
- можно использовать панели управления, включая Hestia и ISPmanager;
- для каждого бокса выдается IPv4 и /64 IPv6;
- дополнительный IPv4 на странице VDS Sprinthost указан как 199 рублей в месяц, а в базе знаний Sprintbox встречается 190 рублей, поэтому перед заказом нужно смотреть актуальную цену в панели.

## Промокод и бонус

Реферальная ссылка для регистрации в Sprintbox: [sprintbox.ru](https://sprintbox.ru/c5568).

Реферальная ссылка для Sprinthost: [sprinthost.ru](https://sprinthost.ru/c5568).

Для Sprintbox / Sprinthost есть бонус по промокоду: клиент получит 219 спринткоинов на баланс аккаунта при первом пополнении.

Сам промокод в заметке пока не указан, поэтому перед публикацией или рекомендацией нужно отдельно добавить код и проверить актуальные условия начисления в кабинете.

## Тестовый бокс Джун+ в Санкт-Петербурге

23 июня 2026 года для проверки Sprintbox был создан бокс в локации Санкт-Петербург на образе Debian 13 "Trixie" x64.

Конфигурация тарифа:

| Параметр | Значение |
| --- | --- |
| Тариф | Джун+ |
| Цена | 400 рублей в месяц |
| CPU | 1 ядро |
| RAM | 1 ГБ |
| Диск | 20 ГБ NVMe |
| IPv4 | 1 адрес включен в тариф |
| IPv6 | можно добавить несколько адресов бесплатно |
| Локация | Санкт-Петербург |

В присланной конфигурации для тарифа Джун+ лимит интернет-канала отдельно не указан, поэтому ниже приведены фактические значения, без пересчета в процент соответствия лимиту.

| Проверка | Результат |
| --- | --- |
| ОС | Debian GNU/Linux 13 (trixie) |
| CPU | Intel Xeon Processor (Skylake, IBRS, no TSX), 1 поток |
| RAM | 967 MiB, swap 4.0 GiB |
| Диск | 20G root-раздел, на момент теста было свободно 9.6G |
| Публичный DNS | около 11.4 ms через `ping`, около 11.5 ms в `mtr`, потерь на конечной точке нет |
| Ya.ru | около 12.3 ms через IPv6 `ping`, около 12.4 ms в `mtr`, потерь на конечной точке нет |
| MTS Москва | IPv4 `iperf3` до московской точки не подключился; IPv6-трасса до сети МТС дошла примерно за 12 ms, но конечная точка не отвечала |

По сети Sprintbox в Санкт-Петербурге выглядит ровно для дешевого тарифа: задержка до российских точек хорошая, Санкт-Петербург в itdoginfo около 3 ms, Москва около 9-12 ms, а ручной `iperf3` по России держится примерно на 95-103 Мбит/с. При этом московский IPv4 `iperf3` не подключился, а upload во Францию почти не прошел, поэтому проблемные направления стоит перепроверить отдельно. Главный слабый результат этого прогона - диск: последовательные операции около 30-34 MiB/s и случайные 4K около 199/88 IOPS выглядят низко даже для дешевого VPS.

## Плюсы

- Дешевые тарифы;
- один личный сайт работает стабильно примерно полгода;
- техническая поддержка отвечает довольно быстро;
- есть реферальная ссылка и бонус по промокоду: 219 спринткоинов при первом пополнении;
- есть тарифные группы Санкт-Петербург, Amsterdam, BY и Frankfurt;
- есть NVMe, бекапы, снапшоты и API;
- есть свой ISO и готовые панели;
- можно управлять PTR;
- есть IPv6;
- в тарифах уже включены 1 IPv4 и возможность добавить несколько IPv6 бесплатно.

## Минусы и риски

- Личного опыта пока мало: один сайт примерно за полгода;
- панель может быть немного непривычной;
- Ташкент, Нью-Йорк, Москва и Владивосток пока готовятся к запуску;
- часть будущих локаций уже видна в панели, но это не значит, что их можно заказать сегодня;
- для Амстердама и других зарубежных локаций скорость в тарифах ниже, чем для Санкт-Петербурга;
- цена дополнительного IPv4 в разных открытых местах отличается, нужно проверять в кабинете;
- у сторонних отзывов по Sprintbox встречаются жалобы на поддержку и сеть, поэтому перед переносом важного проекта нужен тест.

Дополнительный плюс - реферальная ссылка и бонус по промокоду: 219 спринткоинов при первом пополнении.
